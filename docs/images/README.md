# Demo Assets Checklist

Use this checklist to capture screenshots and a short demo GIF.

## Screenshots

Capture these in dark mode with a clean desktop background:

1. **tray.png** — System tray icon + menu open
2. **dashboard-overview.png** — Overview cards + activity graph
3. **dashboard-session.png** — Session expanded with pane list
4. **settings.png** — Settings panel (connection + notifications)
5. **notification.png** — Windows toast notification
6. **error-state.png** — Error banner or disconnected state

## Demo GIF

Create a 30–60s GIF or MP4 demonstrating:

- Launch app from Start Menu
- Tray icon + menu
- Open dashboard
- Trigger a compact/escalation notification

Recommended tools: ScreenToGif, OBS.

## Privacy

- Redact any personal info, API keys, or project names.
- Avoid showing private repositories or usernames.
